GeekItOut is a electronical board game that makes learning fun. You can easily test and learn computer concepts regarding computer fundamentals, hardware and networking. Click START GAME to let the fun begin!

 Rules of the game
  • Every player starts on the USB Block with their USB's
  • With the first round every player get's the chance to randomly choose what category they want to try first
  • Other players get to ask the current player questions that appear on the category chosen
  • If a player answers 1 question right, he get's to move 1 block further
  • If a player answers 2 question right, he get's to move 2 blocks further
  • If a player answers 0 question right, he must move 1 block backwards
  • If a player asnwers 0 questions right on the first round, he get's to choose again a category untill het get's the a question right and get's to move on
